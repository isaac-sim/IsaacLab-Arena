# Cloud-Based CI for IsaacLab Arena

## Usage

### Building and Running the CLI

```sh
cd cloudci && ./build && ./run
```

### Deploying a Github Runner Instance

```sh
cd cloudci && ./deploy
```

### Stopping/Starting a Runner Instance

```sh
cd cloudci && ./stop   # To stop the instance
cd cloudci && ./start  # To start the instance
```

### Destroying the Deployment

```sh
cd cloudci && ./destroy
```
